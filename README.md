# Smart Rental - Property Management System 
 
A complete property rental management platform with MTN MoMo and Airtel Money integration for the Rwandan market. 
 
## Features 
 
### For Renters 
- Search and browse properties 
- Submit rental requests 
- Create and manage invoices 
- Pay rent via MTN MoMo or Airtel Money 
- Chat with landlord 
- Track monthly payment status 
- View payment history and receipts 
 
### For Landlords 
- Dashboard with analytics 
- Manage properties 
- Manage tenants 
- Create rental agreements 
- Track payments received 
- Chat with tenants 
- Monthly payment calculator for each tenant 
- Add notes and track tenant status 
 
### For Admins 
- Platform overview dashboard 
- User management 
- Property management 
- Activity logs viewer 
- Role management

## ?? Environment Variables 
 
Create a .env file in backend folder with: 
``` 
PORT=5000 
DB_HOST=localhost 
DB_USER=root 
DB_PASSWORD=your_password 
DB_NAME=smart_rental_db 
JWT_SECRET=your_jwt_secret 
PAYPACK_CLIENT_ID=your_client_id 
PAYPACK_CLIENT_SECRET=your_client_secret 
```
